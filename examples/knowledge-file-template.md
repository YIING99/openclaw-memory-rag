## Tips for Search Keywords

Search keywords **are for the search engine, not for humans**. When writing them, think:

> "What words might a user search with to find this knowledge?"

Include three categories:
1. **Core concepts**: Tello, eSIM, T-Mobile
2. **Synonyms**: US phone number, virtual number, overseas number
3. **Use-case scenarios**: receive verification code, register AI service, $5 monthly
